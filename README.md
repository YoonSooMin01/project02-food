# Project-Food
어린이 음식 교육용 AI 사진 인식 사이트 


## 🖥️ 프로젝트 소개
이미지 분석을 통해 어린이들이 한식에 흥미를 느낄 수 있도록 만들어 
균형 잡히고 즐거운 식사가 가능하도록 도와주는 사이트 입니다.
<br>

- **Tools** : Spyder (Python 3.10), MariaDB 11.1.2, HeidiSQL 12.3.0.6589, Django, Bootstrap, Tensorflow, Github


- ## 📌 주요 기능

#### Member

- 마이페이지
